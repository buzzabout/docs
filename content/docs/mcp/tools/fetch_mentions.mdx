---
title: fetch_mentions
description: Pull the raw mentions cited by a finished analysis. Returns snippets, platform, and source URLs.
---

`fetch_mentions` returns the actual posts a pattern was distilled from.
This is the citation layer — every claim Claude makes can be traced
back to one of these mentions.

## Input schema

```json title="schema"
{
  "type": "object",
  "required": ["analysis_id"],
  "properties": {
    "analysis_id": { "type": "string" },
    "pattern_id":  { "type": "string", "description": "Optional. Restrict to one pattern." },
    "limit":       { "type": "integer", "default": 20, "maximum": 200 }
  }
}
```

## Example output

```json title="output.json"
{
  "mentions": [
    {
      "id": "men_01H...",
      "platform": "reddit",
      "url": "https://www.reddit.com/r/coffee/comments/...",
      "snippet": "Nobody tells you that nitro cold brew tastes…",
      "author": "u/sipdaily",
      "captured_at": "2026-04-22T14:08:00Z",
      "engagement": { "score": 248, "comments": 32 }
    }
  ],
  "total": 248
}
```

## Notes

- Snippets are truncated to ~280 characters. Open the `url` for the
  full context.
- Mentions are deduplicated across patterns, so the same post never
  appears twice in a single response.
